Resolvers speak JSONL-RPC over stdio. These SDKs handle the framing so you can
focus on the drift logic.

## Shared contract

Both SDKs implement the same protocol, and every message type has a JSON Schema
in [`schemas/`](https://github.com/npupko/hibi/tree/main/schemas) generated from
Hibi's single Zod model (`src/core/model.ts`). That means a resolver in any
language can validate its messages against the same canonical contract.

<Info>
  The Rust SDK builds independently of the CLI:

  ```sh
  cd sdk/rust && cargo build --examples
  ```
</Info>
